# Assets Folder

## Hình Ảnh Sản Phẩm

Vui lòng thêm hình ảnh sản phẩm rượu của bạn vào folder này với tên: **`product.jpg`**

### Khuyến Nghị:

- **Kích thước**: 800x1200px (portrait) hoặc 1200x800px (landscape)
- **Format**: JPG hoặc PNG
- **Chất lượng**: High quality, ít nhất 150 DPI
- **Dung lượng**: Tối ưu dưới 500KB để load nhanh
- **Nội dung**: Hình ảnh chai rượu rõ nét, trên nền sạch hoặc bối cảnh phù hợp

### Cách Thêm:

1. Đổi tên file hình ảnh của bạn thành `product.jpg`
2. Copy file vào folder `assets/`
3. Refresh trang web để xem hình ảnh hiển thị

### Tối Ưu Hình Ảnh (Optional):

Để trang web load nhanh hơn, bạn có thể compress hình ảnh bằng:
- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)
- Photoshop: Save for Web

---

**Lưu ý**: Nếu bạn muốn đổi tên file khác (VD: `my-wine.png`), hãy update đường dẫn trong `index.html`:

```html
<img src="assets/my-wine.png" alt="Rượu Truyền Thống" class="product-image">
```
