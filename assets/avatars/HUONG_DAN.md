# Hướng Dẫn Upload Avatar Cho Testimonials

## Bước 1: Chuẩn bị ảnh avatar
- Chuẩn bị 3 ảnh avatar cho 3 khách hàng
- Kích thước khuyến nghị: **200x200px** hoặc lớn hơn (hình vuông)
- Format: JPG hoặc PNG
- Đặt tên file:
  - `avatar-1.jpg` - Anh Minh Tuấn
  - `avatar-2.jpg` - Chị Lan Anh  
  - `avatar-3.jpg` - Anh Quốc Hùng

## Bước 2: Upload ảnh
1. Mở thư mục: `f:\gravityGoogle_IDE\ladi_ruou_ongtre\assets\avatars\`
2. Copy 3 file ảnh avatar vào thư mục này
3. Đảm bảo tên file đúng như đã liệt kê ở trên

## Bước 3: Kiểm tra
1. Mở file `index.html` trong trình duyệt
2. Scroll xuống phần "Đánh Giá Thực Tế"
3. Bạn sẽ thấy 3 ảnh avatar hiển thị

## Lưu ý:
- Nếu ảnh không hiển thị, kiểm tra lại tên file và đường dẫn
- Ảnh sẽ tự động bo tròn và có viền trắng
- Nếu muốn thay đổi avatar khác, chỉ cần thay file ảnh cùng tên
